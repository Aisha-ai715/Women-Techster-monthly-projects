# Month 4 React Project

This is my first React project created during the Women Techsters Fellowship. I used **Vite** to set up the project because it is fast, modern, and easy for beginners.

## Assignment Questions

### What is Vite?
A frontend build tool that aims to enhance development experience for modern web projects. Vite is a JavaScript bundler. It transforms the code into raw html, css and js so it can be run by browser.

### What is NPM?
It is a package manager for NodeJS. It is the largest single language code repository and a tool for installing and managing packages from the repository on the command line.

### What is package.json?
It is a configuration file used in Node.js projects to manage project metadata, dependencies, and scripts.
